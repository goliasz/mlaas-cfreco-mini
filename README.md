# mlaas-cfreco-mini

Fast in memory item-item recommendation engine based on https://pypi.python.org/pypi/cf_recommender/1.0.1 library.

# Container Startup

```
$ docker run -itd --name=cfreco --hostname=cfreco -p 0.0.0.0:5000:5000 goliasz/mlaas-cfreco-mini
```
